# Assumptions

- SI units are used throughout.
- Blood density and viscosity are constants in v1.
- Vessel segment area is constant in time for the reduced-order solver in v1.
- Pulsatility is modeled by sinusoidal inflow modulation.
- Biomarker field is piecewise constant per segment for v1.
- Sensor samples a mapped local microdomain concentration from bulk concentration.
