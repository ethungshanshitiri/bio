# bloodstream-sim

`bloodstream-sim` is a MATLAB-first framework for reduced-order bloodstream transport simulation with nanomachine agents and biomarker sensing.

## Features
- Directed vascular graph with pulsatile reduced-order flow solver.
- Flow regime classification per segment and time step.
- Local patch interfaces for laminar, transitional, and turbulent corrections.
- Stochastic nanomachine advection-diffusion routing with energy state.
- 1D biomarker advection-diffusion-reaction transport.
- Sensor model with receptor kinetics, transducer output, and binary decisions.
- End-to-end observables and summary metrics.

## Quick start
1. Open MATLAB in this folder.
2. Run `startup`.
3. Run `run_demo`.
4. Run `run_all_tests`.
