Ethungshan Shitiri academic portfolio site

What this is
A minimalist academic portfolio built with plain HTML, CSS, and small JavaScript.
It is ready for GitHub Pages.

Quick deploy
1. Create a GitHub repository
   Recommended name is yourusername.github.io
2. Upload the contents of this folder to the repository root
3. In GitHub, open Settings then Pages
4. Set Source to Deploy from branch
5. Select main branch and root folder
6. Save and wait for the site to publish

Edit content
- Homepage content is in index.html
- News feed is in assets/data/news.json
- Projects are in assets/data/projects.json
- Publications are in assets/data/publications.json
- Social links are in assets/data/links.json

Replace images
- Replace assets/img/profile-placeholder.svg with your photo
  Option A add assets/img/profile.jpg and update index.html img src
  Option B replace the placeholder svg with a real image and keep the name
- Replace assets/img/upc.svg and assets/img/n3cat.svg with official logos if you have permission

CV PDF and publications PDF
- Files are in assets/files
- Replace Ethungshan_Shitiri_CV.pdf with your newest CV if needed

Printing a CV
- Open cv.html and use Print or save PDF
- This version is designed to exclude the news feed
